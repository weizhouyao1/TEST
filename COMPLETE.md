## 🚀 快速部署步骤

### 第1步：配置Firebase (5分钟)
```bash
1. 访问 https://console.firebase.google.com
2. 创建新项目
3. 启用Authentication和Firestore
4. 复制Web应用配置
5. 编辑 js/firebase-config.js，填入你的配置
```

### 第2步：本地测试 (5分钟)
```bash
# 进入项目目录
cd "e:\APP coding"

# 启动本地服务器
python -m http.server 8000

# 访问 http://localhost:8000/login.html
# 注册一个测试账户
```

### 第3步：配置支付（可选）
支付功能框架已准备好，需要：
- 申请支付宝商户账户（https://open.alipay.com）
- 申请微信支付商户账户（https://pay.weixin.qq.com）
- 在支付宝/微信后台获取API密钥
- 集成支付回调处理（需要后端服务）

### 第4步：部署上线
```bash
# 方案A：Firebase Hosting (推荐)
firebase deploy

# 方案B：Netlify
拖拽上传或Git连接

# 方案C：自己的服务器
上传文件到服务器

# 方案D：其他平台
Vercel, AWS S3, Heroku等
```

## 💡 核心特性解析

### 1️⃣ 免费版 vs 付费版
```javascript
免费版 (永久免费)
├─ 100条交易/月
├─ 无投资管理
├─ 基础报表
└─ 社区支持

专业版 (¥9.9/月 或 ¥99/年)
├─ 无限制交易
├─ 投资管理(50个)
├─ 高级报表
└─ 邮件支持

企业版 (¥29.9/月 或 ¥299/年)
├─ 无限投资
├─ 多账户管理
├─ 税务建议
└─ 电话支持
```

### 2️⃣ 7天试用机制
```javascript
新用户自动获得7天专业版体验
├─ 无需绑定支付方式
├─ 完整功能免费试用
├─ 可随时升级或取消
└─ 试用期后自动降级免费版
```

### 3️⃣ 灵活的订阅管理
```javascript
支持用户操作：
├─ 升级套餐 (免费→专业→企业)
├─ 降级套餐 (企业→专业→免费)
├─ 切换周期 (月付→年付)
├─ 取消订阅 (继续使用到期末)
└─ 自动续费管理
```
